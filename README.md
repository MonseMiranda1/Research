Qué significa la extensión de un archivo 

La extensión de un archivo es la parte final del nombre de un archivo que aparece después del punto (.), como las fotos (.jpg). Sirve para indicar qué tipo de archivo es y qué programa puede abrirlo 

Identificación: Indica el tipo de archivo y la aplicación asociada para abrirlo. 

Formato: Representa cómo están codificados los datos (ej. texto, imagen, binario). 

 

¿Para qué sirve la extensión? 

Identificar el tipo de archivo. 

Indicar qué programa puede abrirlo. 

Ayudar al sistema operativo a manejar el archivo correctamente.
